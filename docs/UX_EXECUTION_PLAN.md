# DropZoneSquads UX Execution Plan (v1)

_Date: 2026-03-12_

## Goal
Improve first-session momentum, reduce confusion, and make urgent actions obvious.

## Phase 1 (Immediate: 1-2 days)
1. Add **"Needs Attention"** summary card in Inbox:
   - Pending join requests
   - Unread notifications
   - Unread direct messages
   - Unread squad chats
2. Ensure each item has direct action destination.
3. Keep copy concise and action-first.

## Phase 2 (Short-term: 3-5 days)
1. Add intent-aware routing after onboarding:
   - "Find a Squad" users -> discovery first
   - "Recruit" users -> post squad first
2. Tighten empty states with one-click CTAs.
3. Standardize status chips with shared `status-pill` styles.

## Phase 3 (Next sprint)
1. Guided first-run checklist UI.
2. In-app "What to do next" cards based on state.
3. UX telemetry events for funnel drop-off.

## Success Metrics
- Faster first action after signup.
- Higher conversion from notification -> action completion.
- Fewer "what do I do now" dead ends.
